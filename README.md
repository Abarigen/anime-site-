# ID CARD Store

Адаптивний інтернет-магазин українською мовою з каталогом, акційними наборами, кошиком, оформленням замовлення та адмін-панеллю.

## Запуск

```powershell
npm start
```

Сайт відкриється за адресою:

```text
http://localhost:3000
```

## Адмін-панель

Перейдіть у розділ `Адмін` на сайті.

Стандартний доступ:

```text
login: admin
password: admin123
```

Для продакшну задайте власні значення у `.env` або змінних хостингу:

```env
ADMIN_LOGIN=your-login
ADMIN_PASSWORD=your-strong-password
```

В адмін-панелі можна:

- створювати, редагувати та видаляти категорії;
- додавати нові товари в категорії;
- змінювати назву, опис, ціну, фото, залишок і активність товару;
- переглядати всі замовлення;
- позначати замовлення як `Відправлено`.

Дані магазину зберігаються у `data/store.json`.

## Замовлення

Покупець додає товар у кошик або натискає `Купити зараз`, після чого заповнює:

- ФІО;
- номер телефону;
- відділення Нової пошти;
- метод оплати: повна оплата за реквізитами або оплата при отриманні.

Після успішного оформлення покупець бачить повідомлення:

```text
Замовлення прийнято в роботу. Наш менеджер найближчим часом зв'яжеться для підтвердження.
```

## Telegram

Якщо задані `TELEGRAM_BOT_TOKEN` і `TELEGRAM_CHAT_ID`, сервер додатково надсилатиме нові замовлення у Telegram. Якщо ці змінні не задані, замовлення все одно зберігаються в адмін-панелі.
